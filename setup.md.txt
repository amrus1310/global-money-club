🚀 Global Money Club: Launch InstructionsFollow these steps to move your code from your computer to a live website.1. Local Folder CheckEnsure your project folder contains only these two core files for now:index.html (The landing page)thanks.html (The mission/confirmation page)2. GitHub Desktop (Recommended)Open GitHub Desktop and select File > New Repository.Name: global-money-clubLocal Path: Choose the folder where your index.html is.Click Create Repository.Click Publish repository to send it to your GitHub account.3. Netlify DeploymentLog in to Netlify.Click Add new site > Import an existing project.Select GitHub and authorize it.Search for your global-money-club repository and select it.Leave the "Build settings" empty.Click Deploy site.4. Final VerificationOnce the deploy is finished, click the link Netlify provides.Ensure the "Join the Mission" button on index.html correctly links to thanks.html.If the button doesn't work, check that the code reads: <a href="thanks.html">.
